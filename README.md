📘 WriteFlow(Blog Application)

This project is a full-stack Blogging Management System developed using Django (Backend), Django REST Framework (APIs), and React (Frontend). It allows users to create, read, update, and delete blog posts in a simple and user-friendly interface. The application includes user authentication, where users can register, log in, and securely manage their own blog content.

The backend provides RESTful APIs for handling blog data and user operations, while the React frontend consumes these APIs to display blogs, handle forms, and manage user interactions. The system is designed with a clean structure, making it easy for beginners to understand full-stack development concepts.

Key features include creating and viewing blog posts, editing and deleting posts by the author, and responsive UI components like header, footer, and blog listings. This project is ideal for freshers to demonstrate their understanding of frontend, backend, and API integration in a real-world application.
